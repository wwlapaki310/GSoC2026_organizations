## 一言要約

クラウドネイティブの巨大エコシステムを技術面と運営面の両方で支える財団型コミュニティ。

## 組織概要

CNCF は Kubernetes、Prometheus、Envoy など多数の cloud native プロジェクトを擁する財団である。単なるコード置き場ではなく、ランドスケープ、オブザーバビリティ、メンタリング、運営ドキュメント、コミュニティ形成まで含めて ecosystem を支えるのが特徴だ。

## 主な技術領域

- Prometheus
- Kubernetes
- OpenTelemetry
- Envoy

## 主なトピック

- クラウド
- Cloud Native
- オブザーバビリティ

## GSoC で関わりそうな内容

個別技術の改善だけでなく、コミュニティ支援ツール、ランドスケープ、ドキュメント、観測基盤周辺など、財団ならではのテーマも多い。

## どんな人に向いていそうか

クラウドネイティブ全般に興味がある人、Kubernetes 周辺だけでなく巨大な OSS 生態系の運営にも関心がある人に向いている。
