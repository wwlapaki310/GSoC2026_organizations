## 一言要約

Java のコーディング規約と可読性を自動で支える、静的解析系 OSS の代表格。

## 組織概要

checkstyle は Java コードが一定の規約に沿って読みやすく保たれるよう、自動チェックを行う開発支援ツールである。スタイルガイドの強制だけでなく、なぜ規約が必要かを開発者に意識させる教育的な役割も持っている。
