## 組織概要

API Dash はクロスプラットフォームの API クライアントで、HTTP や GraphQL に加え、AI API の検証やコード生成、MCP 周辺のユースケースにも踏み込んでいる。Postman 系の文脈で理解しやすい一方、生成 AI 時代の開発体験へどこまで広げるかが見どころになっている。
