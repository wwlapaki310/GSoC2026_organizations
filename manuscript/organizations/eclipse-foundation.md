Eclipse Foundation は「The Global Open Source Foundation」を掲げ、robotics、automotive、tools などに取り組むオープンソースコミュニティ。

## 組織概要

Eclipse Foundation は、The Global Open Source Foundation を打ち出して活動する参加機関である。公開されている GSoC metadata からは、robotics、automotive、tools、cloud native java といったテーマに関わる OSS を継続的に育てており、技術面では java、rtos、eclipsejavaide、jakartaee を軸にした開発へ触れやすいことが分かる。
