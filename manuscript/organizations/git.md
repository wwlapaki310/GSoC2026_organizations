## 組織概要

Git は、Fast,Scalable,Distributed revision control system を打ち出して活動する参加機関である。公開されている GSoC metadata からは、version control、dvcs といったテーマに関わる OSS を継続的に育てており、技術面では shell script、git、c language を軸にした開発へ触れやすいことが分かる。

## GSoC で関わりそうな内容

Ideas list と topic tags から見ると、version control、dvcs を入口にした改善提案や機能追加に関わる可能性が高い。既存プロジェクトの拡張、周辺ツールの整備、開発体験の改善などが主な接点になりそうだ。

## どんな人に向いていそうか

version control、dvcs に興味があり、shell script、git、c language といった技術領域を足がかりに OSS 開発へ入りたい人に向いている。テーマ志向でも技術志向でも入り口を作りやすいタイプの参加機関だ。
