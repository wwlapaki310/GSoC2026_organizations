## 一言要約

自由で信頼できるデスクトップ環境を支える、代表的なオープンソースコミュニティのひとつ。

## 組織概要

GNOME Foundation は、個人が自分で制御できるコンピューティング環境を広げることを目指す団体だ。GNOME は Linux デスクトップの代表格として知られ、デスクトップ環境そのものだけでなく、アプリケーション基盤、開発ツール群、配布や実行の仕組みにも関わっている。単に画面を作るプロジェクトではなく、日常的なコンピューティングを自由ソフトウェアで支える大きな生態系として見ると理解しやすい。

## GSoC で関わりそうな内容

GNOME では、デスクトップ本体だけでなく、アプリケーション開発、UI 基盤、周辺ツール、配布まわりなど、複数のレイヤーにまたがるテーマが出てきやすい。見た目の UI 改善だけではなく、基盤の整備や開発者体験の改善にも取り組める余地があると考えられる。

## どんな人に向いていそうか

Linux デスクトップや GUI 開発に興味がある人には特に相性がよい。加えて、C や Rust を使ったアプリケーション基盤、OSS コミュニティ運営、大規模プロジェクトでの協調開発に関心がある人にも向いている。完成した成果が実際のユーザー体験に近い場所へ届きやすい点も魅力だ。
