Open Transit Software Foundation は「Help make public transit better」を掲げ、apps、Transit、travel などに取り組むオープンソースコミュニティ。

## 組織概要

Open Transit Software Foundation は、Help make public transit better を打ち出して活動する参加機関である。公開されている GSoC metadata からは、apps、Transit、travel、bus といったテーマに関わる OSS を継続的に育てており、技術面では android、java、golang、docker を軸にした開発へ触れやすいことが分かる。
