# はじめに

本書は Google Summer of Code 2026 の参加機関をアルファベット順にたどりながら、世の中のオープンソースコミュニティの広がりを日本語で把握するためのガイドです。

各ユニットでは 2 機関ずつ取り上げます。紙面では 1 ページ 2 機関を想定していますが、原稿データとしては電子書籍変換しやすいように、固定ページではなく見出し構造で管理しています。

## 本書の読み方

本書は、184 の参加機関を「どんな領域の OSS に出会えるか」という観点で素早く眺められるように構成しています。気になる機関を見つけたら、そこから公式サイト、Ideas list、コードホスティング先へたどっていただく使い方を想定しています。

掲載している説明文は、GSoC 公式や各団体の英語説明をそのまま訳したものではなく、公開情報をもとに日本語で独自に要約したものです。そのため、実際の応募時には本書だけで完結させず、必ず一次情報も確認してください。

## 注記

- 本書の対象は 2026-04-03 時点で確認できた Google Summer of Code 2026 の accepted organizations 184 件です

- スター数や代表リポジトリは 2026-04-03 時点までに確認できた範囲の参考情報であり、変動する可能性があります

- 調査時点で確認できなかった項目は `不明` と記載しています

# おわりに

Google Summer of Code の参加機関一覧を最後まで眺めると、OSS が単なる Web アプリやライブラリにとどまらず、科学計算、教育、医療、OS、セキュリティ、メディア、地理空間、コンパイラ、ロボティクスまで広がっていることがよく分かります。

本書の価値は、すべての団体を詳説することよりも、「自分がまだ知らない OSS の世界がどこにあるか」を短時間で見渡せることにあります。ここから先は、気になった団体の Ideas list を読み、コードを眺め、コミュニティの空気を感じ、実際に手を動かしていく段階です。

技術書店向けの一冊として、本書が OSS の入り口を広げる地図になれば幸いです。
